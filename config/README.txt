Configuration Files Directory
=============================

This directory contains all project configuration files:

- pyproject.toml          - Python project metadata and build config
- requirements.txt        - Python dependencies  
- uv.lock                 - UV package manager lock file
- .env.*                  - Environment-specific configurations
- .flake8                 - Flake8 linter configuration
- .hintrc                 - Hint configuration
- .pre-commit-config.yaml - Pre-commit hooks configuration

Note: Some tools require certain config files at project root.
Root copies are maintained for tool compatibility.
